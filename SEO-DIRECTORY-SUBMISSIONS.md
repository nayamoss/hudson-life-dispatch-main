## Best Practices

1. **Use Consistent NAP**: Exact same name, address, phone across ALL directories
2. **Save Logins**: Keep spreadsheet with usernames/passwords for updates
3. **Add Photos**: Upload logo and event photos where possible
4. **Complete Profiles Fully**: More complete = better SEO value
5. **Claim Don't Create**: Always search first and claim existing listings
6. **Verify Listings**: Complete verification processes (email, phone, postcard)
7. **Update Regularly**: Return to key directories quarterly to add content

---

## Red Flags to Avoid

- **Don't Pay** for directory submissions (exceptions: chamber memberships)
- **Avoid Spammy Sites**: If directory looks sketchy, skip it
- **No Inconsistent Info**: Use exact NAP every time
- **Don't Create Duplicates**: One listing per directory only
